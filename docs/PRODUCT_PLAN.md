# Product Plan: Dadi Translator

## Product Name

Dadi Translator

## Product Purpose

Dadi Translator helps employees convert Simplified Chinese documents into professional English Word documents quickly and consistently.

## Target Users

- Executive assistants
- AI and IT support staff
- Project coordinators
- Translation support staff
- Department leaders
- Employees who need to understand or share Chinese documents in English

## Main User Problem

Employees often need to translate Chinese documents into English while keeping the document structure clear. Manual translation takes time, and copy-pasting into chat tools can result in inconsistent formatting.

## Core Solution

Dadi Translator provides a simple upload-to-download workflow:

1. Upload Chinese document.
2. Extract document text.
3. Translate Simplified Chinese into English.
4. Preview and edit the result.
5. Download a professional Word document.

## MVP Features

### 1. Document Upload

Users can upload `.docx`, `.doc`, `.txt`, or `.md` files.

### 2. Text Extraction

The system extracts readable text from the uploaded file.

### 3. AI Translation

The system translates Simplified Chinese into professional English using the OpenAI API.

### 4. Editable Preview

Users can revise the title and translation before downloading.

### 5. Word Export

The system generates a `.docx` file with:

- Dadi Coach logo
- Centered title
- One-column English format
- Professional spacing
