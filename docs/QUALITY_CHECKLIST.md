# Dadi Translator Quality Checklist

Use this checklist before accepting or sharing a generated translated document.

## File Upload Check

- [ ] The uploaded file is `.docx`, `.doc`, `.txt`, or `.md`.
- [ ] The source file contains selectable text, not only images.
- [ ] The first line is the actual document title, or the title is manually corrected after translation.
- [ ] The extracted Chinese text appears correctly in the preview.

## Translation Quality Check

- [ ] The English translation preserves the original meaning.
- [ ] No important paragraph, heading, bullet point, or section is missing.
- [ ] Dadi-specific terms are translated consistently.
- [ ] The tone is professional and suitable for business or internal company use.
- [ ] The English text is clear and natural.
- [ ] The system did not add unsupported claims or extra information.

## Formatting Check

- [ ] The output uses one English column only.
- [ ] The English title is centered at the top.
- [ ] The Dadi Coach logo appears correctly in the document header.
- [ ] Section order matches the original document.
- [ ] Numbered lists and bullet lists remain readable.
- [ ] The document has clean spacing and readable font size.
- [ ] The downloaded file name matches the English title.

## Final Review Check

- [ ] The document has been opened in Microsoft Word or WPS for final checking.
- [ ] The title is correct.
- [ ] There are no obvious spelling or grammar issues.
- [ ] The document is ready for internal use or leader review.
